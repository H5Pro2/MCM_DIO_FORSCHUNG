# 09 Regulation

## Aufgabe

Regulation hält DIO handlungsfähig.

Sie wirkt auf:

- Vorsicht
- Mut
- Vertrauen
- Abstand
- Fokus
- Überkopplung
- Denkbelastung
- Reorganisation

## Prinzip

Regulation ist kein hartes Gate.

Sie verändert nicht die Realität und erzeugt keinen eigenen Entry.

Sie liest:

- Weltkontakt
- Sinneswahrnehmung
- MCM-Feld
- Memory
- Thought
- Konsequenz

und moduliert daraus:

- Haltung
- Fokus
- Distanz
- Handlungstiefe
- Reorganisationsbedarf
- Erholung

Beispiel:

```text
Hohe Unsicherheit -> mehr Abstand, mehr Beobachtung, weniger Motorik.
```

Nicht:

```text
Unsicherheit > Wert -> verboten.
```

## Grenze zum MCM-Feld

Das MCM-Feld sagt:

```text
So wirkt es in mir.
```

Regulation sagt:

```text
So gehe ich mit dieser Wirkung um.
```

Wenn das Feld Überkopplung zeigt, darf Regulation Abstand, Dämpfung oder
Reflexion erhöhen. Sie darf daraus aber keine starre Regel bauen.

## Ziel

Mit Erfahrung soll DIO ruhiger werden:

- weniger Reizreaktion
- mehr Formkontakt
- bessere Realitätsprüfung
- weniger Überdenken
- gezieltere Handlung
