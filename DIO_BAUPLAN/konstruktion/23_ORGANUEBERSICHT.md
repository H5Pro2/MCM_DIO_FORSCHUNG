# 23 Organübersicht

Diese Datei hält die organische Übersicht fest.

Sie ist keine neue Mechanik, sondern eine Orientierung, welche Teile DIO als
Funktionsorgane besitzt.

## Organe im aktuellen Bauplan

## Verbindliche Organ-Kopplung

```text
Sehen   = Form / Struktur
Fühlen  = MCM-Feldwirkung
Hören   = Energie / Frequenz / Marktspannung
Denken  = Hypothese / Verdichtung
Handeln = erst nach tragender Kopplung
```

Kein Organ handelt allein. Sehen, Hören, Fühlen und Denken liefern
unterschiedliche Wahrnehmungs- und Verarbeitungslagen. Das Handlungssystem
darf erst wirksam werden, wenn diese Lagen mit Memory und Realitätsprüfung
tragend gekoppelt sind.

### Weltaufnahme

Nimmt reale Außenwelt auf:

- OHLCV
- Zeit
- Verlauf
- reale Konsequenz

### Visuelles Organ

Bildet Form:

- Richtung
- Verdichtung
- Bereich
- Wiederkehr
- zeitliche Tiefe

### MCM-Feld

Trägt innere Wirkung:

- Spannung
- Kohärenz
- Nachhall
- Distanz
- Tragfähigkeit

### MCM-Neuronen

Lokale Wirkpunkte im Feld:

- Aktivität
- Dämpfung
- Resonanz
- Reorganisation
- Plastizität
- auditive Stimulation als eigene Hoerspur, ohne Entry-Erlaubnis

### Neurochemie

Moduliert Feld und Reaktion:

- Aktivierung
- Schutz
- Stabilität
- Hemmung
- Lernspannung
- Vertrauen

### Memory

Speichert reale Erfahrung:

- Form
- Feldwirkung
- Handlung oder Nicht-Handlung
- Konsequenz
- Reife

### Thought

Bildet Hypothesen als Wahrnehmung des eigenen Denkens:

- Varianten
- Gedankenkeime
- Replay
- Was-wäre-wenn
- offene Möglichkeiten

### Kortex

Organisiert bewusste Verarbeitung:

- Fokus
- Distanz
- Vergleich
- Realitätsprüfung
- Reorganisation

### Handlungssystem

Setzt reale Aktionen:

- Entry
- SL
- TP
- Order
- Cancel
- Nicht-Handlung

### Konsequenzsystem

Führt Realität zurück:

- TP
- SL
- Cancel
- Beobachtungsergebnis
- Positionslast
- Reifegewinn
- Reorganisation

## Wichtig

Kein Organ soll allein handeln.

Handlung entsteht erst aus dem Zusammenspiel:

```text
Sehen -> MCM-Feld -> Memory -> Thought -> Realitätsprüfung -> Handlung
```
