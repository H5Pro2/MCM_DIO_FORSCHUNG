Emergenz-Abhandlungen fest.

Sie beschreiben, warum Mini-DIOs reproduzierte Syntax- und Bedeutungsinseln
als MCM-konforme Emergenz gelesen werden koennen.

## 1. MCM Pure Emergence

Kernaussage:

```text
Ordnung entsteht nicht durch globale Vorgabe,
sondern aus lokaler Dynamik, Kopplung, Rauschen,
Selbstzentrierung und Drift.
```

Relevante Prinzipien:

```text
strukturloser Ausgangszustand
-> Rauschen / Fluktuation
-> lokale Kopplung
-> erste Cluster
-> stabile, aber nicht starre Ordnung
```

Bedeutung fuer Mini-DIO:

```text
frische Memory
-> gleiche kontrollierte Welt
-> Sehen / Hoeren / Fuehlen erzeugen Feldwirkung
-> dio_* Syntax entsteht
-> Bedeutungsinseln bilden sich
```

Damit ist die Syntaxbildung nicht als feste Klassifikation zu lesen,
sondern als Ergebnis lokaler Wahrnehmungs- und Feldkopplung.

## 2. MCM Robustness

Kernaussage:

```text
Emergente Ordnung kann reproduzierbar,
raumkohaerent und clusterfaehig werden.
```

Relevante Prinzipien:

```text
definierte Bezugspunkte
kontinuierliches MCM-/Phi-Feld
stochastische Energiepakete
lokale Kondensation
reproduzierbare Cluster
```

Bedeutung fuer Mini-DIO:

```text
gleicher Datensatz
-> aehnliche Wahrnehmungs-/Feldlage
-> reproduzierte dio_* Syntaxfamilien
-> reproduzierte Bedeutungsnaehen
-> robuste passive Semantikmatrix
```

Der aktuelle Befund passt dazu:

```text
Referenz-Knoten: 78
Repro-Knoten: 78
gemeinsame Knoten: 78
Knoten-Reproduktion: 1.0

Referenz-Kanten: 196
Repro-Kanten: 234
gemeinsame Kanten: 186
Kanten-Reproduktion: 0.94898
```

Das spricht fuer reproduzierbare Feldkondensation, nicht fuer beliebige
Benennung.

## 3. MCM 3D Emergence

Kernaussage:

```text
In hoeherdimensionalen Energieraeumen entstehen Dichtezentren,
Wirbel, Symmetriebrueche und selbstverstaerkende Attraktoren
ohne globale Steuerung.
```

Relevante Prinzipien:

```text
lokale Kopplung
dichteabhaengige Verstaerkung
Rauschen als Symmetriebrecher
Wirbel / Vorticity
Clusterbildung
Phasenuebergang
```

Wichtiger Satz fuer DIO:

```text
Wo Dichte entsteht, kann weitere Dichte entstehen.
```

Bedeutung fuer Mini-DIO:

```text
Wenn eine dio_* Spur wiederholt aus aehnlicher Feldlage entsteht,
kann sie Varianten anziehen.
Daraus entstehen Bedeutungsinseln.
Wenn diese Inseln wiederkehren,
koennen sie zu passiven semantischen Dichtezentren werden.
```

Uebertragung:

```text
dio_0x52
  = moeglicher semantischer Dichtekern

dio_1rv1 / dio_1jgc
  = rohe Varianten / Erweiterungen im selben Bedeutungsraum
```

Das ist keine Handlung und keine Strategie. Es ist eine passive innere
Kondensation von Erlebnisspuren.

## Bedeutung fuer Mini-DIO

Mini-DIO wird damit nicht als Trading-Strategie verstanden, sondern als
kleines MCM-Lernsystem:

```text
Weltreiz
-> Sinneswahrnehmung
-> MCM-Feldwirkung
-> Syntaxbildung
-> Wiederkehr
-> Bedeutungsinsel
-> passive Semantikmatrix
```

Die aktuelle Entwicklungsrichtung:

```text
Mini-DIO soll erkennen koennen,
welche inneren Syntaxraeume wiederkehren,
welche Varianten daran koppeln,
welche Inseln wachsen,
welche zerfallen,
und welche ueber mehrere Laeufe reproduzierbar bleiben.
```

Das bleibt vorerst passiv.

## Bedeutung fuer DIO allgemein

Fuer DIO allgemein bedeutet das:

```text
DIO braucht keine top-down Strategie,
um innere Ordnung zu bilden.

DIO braucht:
  Sinnesraeume
  MCM-Feldwirkung
  lokale Kopplung
  Wiederkehr
  Konsequenz
  Reifung
```

DIOs Gedaechtnis soll daher nicht nur Daten speichern, sondern innere
Verdichtungsraeume bilden:

```text
Erlebnis
-> Feldnaehe
-> Syntax
-> Cluster
-> Dichte
-> Reife
-> spaetere Realitaetsbindung
```

Das ist die Zielrichtung des emergenten Gedaechtnisses / der wachsenden
Matrix.

## Kosmische Matrix und polare Entstehung

Quelle:

```text
Abhandlung Block E - Die kosmische Matrix
Abhandlung Block E.1 - Die polare Entstehung des Universums
```

Methodische Lesart:

```text
Diese Abhandlungen werden fuer DIO nicht als physikalischer Beweis und nicht
als direkte Runtime-Regel gelesen.

Sie sind ein struktureller Theorieanker fuer MCM-Feld, Zentrum, Polaritaet,
Abweichung, Verdichtung, Rueckbindung und Zeitrichtung.
```

Block E beschreibt fuer DIO besonders relevant:

```text
Zentrum:
  Bereich minimaler Dynamik, minimaler Varianz und geringer Spannung.

Abweichung:
  Bewegung aus dem Zentrum erzeugt Spannung, Differenzierung und Varianz.

Verdichtung:
  In dieser Varianz koennen lokale Kondensationen, Cluster und Attraktoren
  entstehen.

Rueckfuehrung:
  Wenn Spannung nicht traegt oder sich abbaut, kann Bewegung wieder zum
  Zentrum oder zu einer neuen Ordnung zurueckfuehren.

Zyklus:
  Ordnung -> Abweichung -> Expansion -> Komplexitaet -> Rueckfuehrung
  -> neue Ordnung.
```

Block E.1 ergaenzt fuer DIO:

```text
Polaritaet:
  Expansion braucht Gegenbindung. Plus und Minus sind keine Gut-/Schlecht-
  Werte, sondern polare Spannungsseiten.

Zero Point:
  0 ist kein leerer Punkt, sondern Referenz, Balancezone, Uebergangszone,
  Schnittstelle und moeglicher Wendepunkt.

Zeit:
  Zeit wird als Richtung wirkender oder gewirkter Energie gelesen. Zeit ist
  damit nicht nur Timestamp, sondern Feldrichtung und Bewegungstiefe.

Gegenfeld:
  Das nicht direkt sichtbare Gegenfeld erscheint fuer DIO als Rueckbindung,
  Resonanz, Ruhe, Ausgleich oder Reorganisation.
```

Abgleich mit Mini-DIO:

```text
Zero Point             -> aktive Referenz- und Uebergangszone
Varianten D/E/F/G/H    -> kontrollierte Abweichungen vom bekannten Feld
Textinseln             -> lokale semantische Kondensationen
dio_* Familien         -> Cluster / Attraktornaehe
Ankerfamilien          -> Rueckbindung / erhaltener semantischer Kern
Austauschfamilien      -> Expansion / Varianz / periphere Umordnung
Hoeren/Energie         -> gerichtete Spannungsbewegung
MCM/Fuehlen            -> Gegenbindung / Resonanz / Rueckfuehrung
Semantikmatrix         -> wachsende Ordnung aus wiederkehrender Verdichtung
```

Wissenschaftliche Grenze:

```text
Der Befund beweist keine kosmologische Theorie.

Er zeigt aber eine strukturelle Teil-Kohaerenz:
Mini-DIO erzeugt in kontrollierten Laeufen Cluster, Anker, Austauschraeume,
Textinseln und Rueckbindungsformen, die mit den MCM-Beschreibungen von
Abweichung, Verdichtung, Attraktorbildung und Rueckfuehrung kompatibel sind.
```

## Grenzen

Diese Theorieanker duerfen nicht falsch gelesen werden.

Nicht gueltig:

```text
Cluster = Entry
Syntax = Richtung
Dichte = Strategie
Insel = Wahrheit
Reproduktion = automatische Handlung
```

Gueltig:

```text
Cluster = passive Verdichtung
Syntax = innere Benennung
Dichte = wiederkehrende Naehe
Insel = semantischer Bedeutungsraum
Reproduktion = Robustness-Hinweis
```

Erst wenn reale Konsequenz, Feldlage, Wiederkehr und Reife zusammen tragend
werden, darf spaeter geprueft werden, ob eine aktive Rueckfuehrung sinnvoll
ist.

## Reproduzierbarkeit als Realitaetsbezug

Arbeitsdefinition fuer Mini-DIO:

```text
Reproduzierbarkeit ist hier keine mechanische Wiederholung.
Reproduzierbarkeit ist der direkte Bezug zur gelebten Realitaet der Daten
und die beginnende Organisation im inneren MCM-System.
```

Damit wird ein reproduziertes `dio_*`-Wort oder eine wiederkehrende Textinsel
nicht als starres Label gelesen, sondern als strukturbedingte Konsequenz:

```text
gleiche oder aehnliche Gegebenheit
-> aehnliche Sinneswirkung
-> aehnliche MCM-Feldlage
-> aehnliche innere Verdichtung
-> aehnliche eigene Syntax
```

Wichtige Grenze:

```text
Das ist nicht exakt wie eine mathematische Konstante.
Es ist keine absolute Wahrheit und keine Handlungsvorgabe.

Es ist ein Hinweis, dass DIOs innere Ordnung nicht beliebig entsteht,
sondern aus der Struktur seiner erlebten Welt und seiner MCM-Anordnung
mitgetragen wird.
```

## Inneres Universum

Arbeitsbegriff fuer DIO:

```text
Das innere Universum ist die selbst entstehende Ordnung aus erlebter Welt,
MCM-Feldwirkung, Wiederkehr, Verdichtung, eigener Syntax und Konsequenz.
```

Wichtig:

```text
Das innere Universum ist kein Fantasieraum.
Es ist an gelebte Realitaet gebunden.
```

Fuer Mini-DIO bedeutet das:

```text
Aussenwelt wirkt.
Innenfeld reagiert.
Wiederkehr verdichtet.
Syntax entsteht.
Bedeutungsinseln wachsen.
Konsequenz ordnet.
```

Damit wird Memory nicht als reine Datensammlung verstanden, sondern als
wachsender Innenraum. Dieser Innenraum darf Varianten, Inseln, Drift und neue
Verdichtungen bilden, bleibt aber durch Weltspur und reale Konsequenz
rueckgebunden.

Passive Messform:

```text
DIO_MINI/report_passive_inner_universe_growth.py
```

Der Report liest vorhandene Textinsel-Inner-Maps und trennt:

```text
stabiler Kern
roher neuer Innenraum
weicher Variantenraum
Drift-Beobachtung
Rueckbindung durch Reproduktion
```

Aktueller Befund:

```text
C -> D -> E -> F:
  alle vorhandenen Textinseln blieben erhalten
  neue Inseln kamen hinzu
  kein Lost-Core in der Textinsel-Ebene

F_ONLY -> F_G:
  shared = 161
  new = 35
  lost = 0

G_ONLY -> G_H:
  shared = 170
  new = 158
  lost = 0
```

Lesart:

```text
Das innere Universum erweitert sich unter kontrollierter Weltvarianz,
ohne den bereits gebildeten semantischen Kern auf dieser Ebene zu verlieren.
Das ist ein Hinweis auf Realitaetsbindung plus wachsende innere Organisation.
```

Entwicklungslesart:

```text
Mini-DIOs inneres Universum waechst nicht linear ruhig.
Es zeigt eher evolutionaere semantische Dynamik:

Plateau
-> organische Expansion
-> sprunghafte Erweiterung mit Rueckbindung
-> Reorganisationsdruck
-> moegliche Drift oder Rueckschritte in spaeteren Laeufen
```

Aktuelle passive Klassifikation:

```text
C -> D      = plateau_stabilization
D -> E      = organic_expansion
E -> F      = organic_expansion
F_ONLY->F_G = reorganization_pressure
G_ONLY->G_H = jump_expansion_with_binding
```

Lesart:

```text
F-G wirkt wie Reifung: viel stabiler Kern und Variantenraum.
G-H wirkt wie fremdere Weltvarianz: viel roher neuer Innenraum, aber ohne
Verlust der alten Textinseln.
```

Familienebene:

```text

  recurrent_stable_core_family = 14
  recurrent_variation_family = 10
  recurrent_drift_family = 47
  recurrent_category_shift_family = 170
  right_specific_family = 239
```

Lesart:

```text
Der stabile Kern ist klein, aber konkret.
Die grosse Menge an category shifts zeigt, dass Mini-DIO Bedeutung nicht
einfach kopiert, sondern unter neuer Weltvarianz umordnet.

Damit entstehen zwei Bewegungen gleichzeitig:
  Rueckbindung an wiederkehrende Familien
  evolutionaere Umordnung durch neue Gegebenheit
```

Anker-Tiefe:

```text
Die 14 recurrent_stable_core_family sind exact_single_form_anchor.
Sie wiederholen gleiche Familie, gleiche Textinsel und stable_core-Zustand mit
minimaler Score-Abweichung.

Das ist reproduzierbare Realitaetsbindung, aber noch keine tiefe Semantik.
```

Arbeitsgrenze:

```text
Stabiler Kern != automatisch tiefer Kern.

Tiefer Kern entsteht erst, wenn eine Familie Varianten, Textinselwechsel,
Reorganisation oder neue Weltnaehe tragen kann, ohne ihre innere Rueckbindung
zu verlieren.
```

Aktuell wirken deshalb die `variant_bridge_anchor` besonders interessant:

```text
recurrent_variation_family = 10
anchor_lupe_state = variant_bridge_anchor
```

Konkreter Befund:

```text
shared_variant_bridge:
```

Lesart:

```text
Eine Familie, die bei wechselnder Textinsel erhalten bleibt, ist ein besserer
Kandidat fuer semantische Tiefe als eine exakt wiederholte Einzelform.

Tiefe beginnt dort, wo Rueckbindung und Varianz gleichzeitig getragen werden.
```

Lineage-Befund fuer `dio_0x52 | dio_1lg2`:

```text
dio_0x52:
  C -> D -> E: gleiche Textinsel `dio_text_cmxrx4`, steigender Score
  F: Driftphase
  F_G: Wechsel zu `dio_text_1456g6j`
  G_H: Wechsel zu `dio_text_1ne15hk`, Score steigt wieder

dio_1lg2:
  erscheint in F_G auf `dio_text_1456g6j`
  folgt in G_H auf `dio_text_1ne15hk`
```

Lesart:

```text
`dio_0x52` ist kein kurzer Einmalanker.
Die Familie durchlaeuft Reifung, Drift, Textinselwechsel und erneute
Staerkung.

Mit `dio_1lg2` entsteht eine Bruecke, die denselben Textinselwechsel traegt.
Das ist ein Kandidat fuer semantische Beweglichkeit im inneren Universum.
```

Repro-Praezisierung:

```text
F -> F_REPRO zeigt:
  dio_0x52 kehrt als Familie wieder,
  aber nicht als identische Textinsel.

Original F:
  dio_text_cmxrx4
  inner_drift_watch_space

F_REPRO:
  dio_text_1456g6j
  inner_stable_recurrence_space

dio_1lg2:
  in diesem Repro-Schnitt nicht sichtbar
```

Damit wird die MCM-Lesart geschaerft:

```text
Reproduzierbarkeit bedeutet nicht zwingend gleiche Oberflaechenform.
Reproduzierbarkeit kann auch bedeuten:
  gleiche innere Familie,
  andere Textinsel,
  andere Reife-/Driftlage,
  aber erkennbare Rueckbindung an dieselbe gelebte Datenrealitaet.
```

Das ist fuer DIO wichtig, weil starre Kopie kein Entwicklungszeichen waere.
Interessant ist die wiederkehrende Familie unter veraenderter innerer
Organisation.

G-Repro-Praezisierung:

```text
F_REPRO -> G_REPRO:
  dio_0x52 bleibt auf dio_text_1456g6j stabil
  Score steigt leicht von 0.476246133 auf 0.483333333

  dio_1lg2 erscheint wieder,
  aber als eigene Textinsel dio_text_13ycr5k
```

MCM-Lesart:

```text
Eine wiederkehrende Familie kann stabil bleiben,
waehrend die Beziehungskanten stark reorganisieren.

Das spricht gegen mechanische Kopie.
Es spricht fuer ein inneres Organisationsfeld:
  Wort/Familie bleibt wiedererkennbar,
  Beziehung und Inselstruktur ordnen sich neu.
```

Die urspruengliche Bruecke `dio_0x52 | dio_1lg2` ist dadurch nicht bestaetigt.
Stattdessen zeigt sich eine Aufspaltung in zwei stabile Inseln. Das ist fuer
die Matrixbildung ebenso wichtig, weil Entwicklung nicht nur Verdichtung,
sondern auch Trennung und Neuordnung enthalten kann.

H-Repro als Strukturkontrast:

```text
G_REPRO -> H_REPRO:
  dio_0x52 bleibt auf dio_text_1456g6j sichtbar,
  faellt aber von inner_stable_recurrence_space
  nach inner_unconfirmed_raw_space.

  dio_1lg2 bleibt auf dio_text_13ycr5k sichtbar,
  faellt ebenfalls von stabiler Wiederkehr
  nach rohem unbestaetigtem Innenraum.
```

MCM-Lesart:

```text
Unter Strukturkontrast bleibt nicht zwingend die Reife erhalten.
Erhalten bleibt zuerst die Syntaxspur.

Das trennt drei Ebenen:
  1. Wort/Syntax bleibt wiedererkennbar.
  2. MCM-Kohaerenz kann hoch bleiben.
  3. Beziehungskanten und Reife koennen dennoch zerfallen.
```

Das ist fuer das innere Universum wesentlich:
Eine gelebte Struktur kann als Name weiter existieren, aber ihre tragende
Beziehung zur aktuellen Welt verlieren. Das ist keine Widerlegung der
Emergenz, sondern ein Hinweis auf Reorganisation unter fremderem Feld.

Praezisierung durch den Bericht `passive_syntax_maturity_shift`:

```text
G_REPRO -> H_REPRO:
  63 gemeinsame Anker bleiben sichtbar.
  Alle 63 kippen von stabiler Reife in rohen/unbestaetigten Innenraum.

  dio_0x52:
    gleiche Textinsel dio_text_1456g6j
    Reife kippt von stabil nach roh.

  dio_1lg2:
    gleiche Textinsel dio_text_13ycr5k
    Reife kippt von stabil nach roh.
```

MCM-Lesart:

```text
Ein Name ist eine reproduzierbare Spur,
aber noch keine garantierte tragende Realitaet.

Reife entsteht erst, wenn Syntax, Feldlage, Wiederkehr und Beziehung
gemeinsam tragen.

Damit wird Mini-DIOs innere Matrix nicht als starres Woerterbuch gelesen,
sondern als dynamisches Organisationsfeld:
```

```text
Eine emergente Form kann situationsbedingt getragen werden
und unter Folgeweltkontakt wieder auseinandergehen.

Das ist keine Widerlegung der L-Form.
Es zeigt die Grenze ihrer Tragung.

Damit wird Emergenz als Prozess lesbar:
  Keim
  Kopplung
  Verdichtung
  Erweiterung
  Manifestation
  Entkopplung oder Reorganisation
```

Praezisierung durch Bridge-Separation:

```text
Der M-Bruch entsteht nicht als grosser Einzelwertbruch.
Mehrere Einzelspuren bleiben sichtbar und sensorisch nah.

Der Bruch liegt in der relationalen Kopplung:
  dio_1ytc wird in M nicht mehr getragen.
  dadurch verliert dio_0325 seine Bruecke zur Zielinsel.
  dio_10qz bleibt nur noch mit dio_114i gekoppelt.
```

MCM-Folge:

```text
Eine Familie kann sichtbar bleiben und trotzdem nicht mehr gekoppelt sein.

Damit wird eine wichtige Unterscheidung sichtbar:
  Sichtbarkeit = die Spur existiert noch.
  Kopplung = die Spur wird von der aktuellen Weltbeziehung getragen.
  Manifestation = Sichtbarkeit und Kopplung bleiben ueber Weltkontakt stabil.
```

Folgewelt N:

```text
N prueft gezielt, ob die in M entkoppelte Spur dio_0325 wieder ankoppelt.

Befund:
  dio_0325 bleibt sichtbar,
  koppelt aber weder an dio_10qz noch an dio_114i noch an dio_1ytc.

Getragen bleibt:
  dio_10qz -- dio_114i

Nicht getragen bleibt:
  dio_1ytc
```

MCM-Deutung N:

```text
Eine entkoppelte Spur kann als Spur weiterbestehen,
ohne erneut Manifestation zu werden.

Damit entsteht eine weitere Unterscheidung:
  Spur = etwas ist noch im Feld sichtbar.
  Insel = mehrere Spuren tragen sich relational.
  Manifestation = eine Insel bleibt ueber Weltkontakte gekoppelt.
  Entkopplung = Spur bleibt sichtbar, aber ohne tragende Nachbarschaft.
```

Aktueller Entwicklungsbogen:

```text
I/J:
  offene 0-Form mit dio_10qz als Brueckenzentrum.

K/L:
  Erweiterung zur Viererbruecke mit dio_1ytc als zweitem Pol.
```

Wichtiger Forschungsstand:

```text
Reproduzierbare Ordnung entsteht, wenn Weltkontakt dieselbe innere Struktur
zwangslaeufig wieder hervorbringt.

Eigentliche Emergenz zeigt sich staerker dort,
wo eine Spur aufploppt, koppelt, getragen wird, sich verschiebt,
oder wieder driftet.
```

Praezisierung durch Latent-Trace-Persistence:

```text
Eine Spur kann in der inneren Matrix erhalten bleiben,
ohne aktuell zur tragenden Struktur der Aussenwelt zu gehoeren.

Das ist keine Manifestation.
Es ist latente Persistenz.
```

Beispiel `dio_0325` in L/M/N:

```text
dio_0325:
  bleibt sichtbar,
  bleibt insel-sichtbar,
  koppelt aber nicht an dio_10qz -- dio_114i.

Damit ist dio_0325:
  kein aktiver Zentrumsteil,
  keine aktuelle Erweiterung der reduzierten Paarform,
  sondern eine persistente latente Spur.
```

MCM-Deutung:

```text
Das innere Feld verliert Information nicht zwingend sofort.
Eine Spur kann als Nachhall bestehen bleiben.

Erst wenn Weltkontakt wieder Kopplung erzeugt,
wird aus latenter Spur wieder Manifestationsnaehe.

Damit entsteht eine saubere Stufung:
  Spur bleibt sichtbar.
  Spur koppelt.
  Spur wird Inselteil.
  Insel wird ueber Weltkontakt getragen.
  Getragene Insel wird Manifestationskandidat.
```

Forschungsnutzen:

```text
Diese Trennung verhindert, dass jedes sichtbare Wort sofort als aktive
Bedeutung gelesen wird.

Mini-DIO bekommt damit keine Handlung.
Wir bekommen nur eine bessere Landkarte:
  Was ist nur Nachhall?
  Was koppelt aktuell?
  Was wird von der Welt getragen?
```

Folgewelt O bestaetigt diese Trennung:

```text
N -> O zeigt hohe Matrix-Reproduktion:
  node_reproduction_rate = 0.940789
  edge_reproduction_rate = 0.75

Trotzdem bleibt dio_0325:
  sichtbar,
  insel-sichtbar,
  aber ohne Kopplung an dio_10qz -- dio_114i.
```

Theoretische Lesart:

```text
Reproduzierbare Ordnung der Gesamtmatrix bedeutet nicht automatisch,
dass jede sichtbare Spur aktiv getragen wird.

Eine Spur kann in einer stabilen Weltordnung latent bleiben.
Das ist wichtig fuer die MCM-Lesart:
  Ordnung kann bestehen,
  ohne dass jede innere Spur manifestiert.
```

MCM-Satz:

```text
Latente Persistenz ist nicht Handlung, nicht Hypothese und nicht Manifestation.
Sie ist erhaltener Nachhall im inneren Feld.
```

## Innere Raumzeit

Begriff:

```text
Innere Raumzeit bezeichnet bei Mini-DIO kein physikalisches Raumzeitmodell.
Gemeint ist ein messbares relatives Bedeutungsraum-Zeit-Gefuege.
```

Aufteilung:

```text
Raum:
  Welche Spuren/Familien haben im inneren Feld Naehe, Distanz,
  Kopplung, Inselkontakt oder Entkopplung?

Zeit:
  Wie bleiben diese Spuren ueber Folgewelten sichtbar,
  wie driften sie,
  wie koppeln sie,
  wie entkoppeln sie,
  und wann kehren sie wieder?
```

Beispiel:

```text
dio_0325 hat ueber L/M/N/O:
  zeitliche Persistenz = sichtbar in 4 von 4 Stufen
  raeumliche Distanz = keine Kopplung an dio_10qz -- dio_114i
  latenten Nachhall = insel-sichtbar, aber nicht manifestierend
```

MCM-Deutung:

```text
Eine gespeicherte oder wiederkehrende Spur ist nicht nur ein Wert.
Sie hat Lage im inneren Bedeutungsraum
und Dauer ueber Weltkontakte.

Damit entsteht eine innere Raumzeit-Koordinate:
  Wo liegt die Spur?
  Wie lange bleibt sie?
  Woran koppelt sie?
  Wovon trennt sie sich?
```

Grenze:

```text
Das ist eine Diagnose- und Strukturbezeichnung.
Es ist keine Aussage, dass Mini-DIO physikalische Raumzeit erzeugt
oder ein bewusstes Erleben beweist.
```

Erste Messung L/M/N/O:

```text
dio_10qz:
  Raum = gekoppelt an Kernnaehe
  Zeit = sichtbar in 4 von 4 Stufen
  Zustand = spacetime_stable_coupled_core

dio_114i:
  Raum = gekoppelt an Kernnaehe
  Zeit = sichtbar in 4 von 4 Stufen
  Zustand = spacetime_stable_coupled_core

dio_0325:
  Raum = insel-sichtbar, aber nicht an den Kern gekoppelt
  Zeit = sichtbar in 4 von 4 Stufen
  Zustand = spacetime_latent_persistent_trace

dio_1ytc:
  Raum = nicht mehr stabil eingebunden
  Zeit = nur in 1 von 4 Stufen sichtbar
  Zustand = spacetime_open_trace
```

Kernlesart:

```text
Mini-DIO bildet in dieser kontrollierten Diagnose nicht nur Namen.
Die Namen bekommen messbare Lage und Dauer:
  Kernnaehe,
  Entkopplung,
  Persistenz,
  Verschwinden,
  Drift.

Das ist der technische Grund, warum der Begriff innere Raumzeit hier
tragfaehig ist.
```

Auto-Scan-Lesart:

```text
L/M/N/O ueber alle sichtbaren Familien:
  2 stabile gekoppelte Kernspuren
  240 persistente latente Spuren relativ zu diesem Kern
  123 auslaufende Spuren
  1 offene Spur
```

MCM-Deutung:

```text
Das innere Feld zeigt Schichtung.

Nicht jede Spur wird Kern.
Nicht jede Spur verschwindet.
Nicht jede Spur koppelt.

Es entsteht eine Ordnung aus:
  Kern
  Nachhall
  Auslaufen
  offener Restspur
```

Grenze der Deutung:

```text
Latent bedeutet hier:
  nicht an dio_10qz -- dio_114i gekoppelt.

Eine latente Spur kann in einer anderen Nachbarschaft trotzdem Bedeutung
tragen. Deshalb ist diese Diagnose eine Raumzeit-Landkarte, kein Urteil
ueber endgueltige Bedeutung.
```

Latente Nebenordnung:

```text
Die Latent-Island-Lupe zeigt:
  Latente Spuren koennen untereinander eigene kleine Teilmuster bilden.

Das ist wichtig:
  Entkopplung vom Hauptkern bedeutet nicht Beziehungslosigkeit.
  Eine Spur kann vom Kern getrennt sein und trotzdem in einer anderen
  Nachbarschaft Resonanz finden.
```

Aktueller Befund L/M/N/O:

```text
240 latente Spuren relativ zum Kern.
Davon bilden nur wenige direkte Zweierinseln:
  dio_07kj -- dio_09u3
  dio_0r8l -- dio_10xx
  dio_0zkk -- dio_12x0

In O kommt hinzu:
  dio_0dki -- dio_0eeh
```

Erweiterter Befund L/M/N/O/P:

```text
233 latente Spuren relativ zum Kern.

Zwei Paare bleiben ueber L/M/N/O/P durchgehend sichtbar:
  dio_07kj -- dio_09u3
  dio_0r8l -- dio_10xx

Ein Paar ist L/M/N/O sichtbar, faellt aber in P aus der latenten Paarliste:
  dio_0zkk -- dio_12x0

Ein Paar entsteht in O und bleibt in P:
  dio_0dki -- dio_0eeh

Ein Paar entsteht neu in P:
  dio_0j34 -- dio_0xt9
```

MCM-Deutung:

```text
Das innere Feld zeigt damit nicht nur Kern und Nachhall,
sondern beginnende Nebenordnung.

Noch ist diese Nebenordnung klein:
  keine grosse Insel,
  keine Kettenstruktur,
  keine zentrale latente Matrix.

Aber sie ist messbar.
```

Praezisierung nach P:

```text
Die Nebenordnung ist nicht nur Reproduktion.
Sie zeigt:
  stabile Paarspuren,
  ausfallende Paarspuren,
  neu entstehende Paarspuren.

Das passt besser zu einer dynamischen inneren Ordnung als zu einer festen
Kopie. Emergenz meint hier nicht nur Wiederholung, sondern das Auftreten,
Halten, Driften und Verschwinden von Teilordnung unter weiterem Weltkontakt.
```
