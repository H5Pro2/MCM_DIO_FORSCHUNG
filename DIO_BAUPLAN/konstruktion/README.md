# DIO Konstruktion

Dieser Ordner ist der neue textliche Aufbau von DIO.

Ziel ist ein sauberer Schnitt:

- weniger alte Begriffe
- weniger überladene Pläne
- klare Schichten
- klare Zuständigkeiten
- keine versteckte Wenn-Dann-Mechanik
- Hypothesen bleiben Gedanken, nicht Realität
- MCM bleibt Gefühlswahrnehmung, nicht mechanische Strategielogik
- Strategie darf entstehen, wird aber nicht erzwungen

Die alte Dokumentation bleibt als Historie erhalten. Dieser Ordner ist ab
jetzt der aktive Bauplan und spätere Konstruktionsplan, mit dem DIO aufgebaut
und zurückgeführt wird.

Erkenntnis- und Theorieabhandlungen liegen bewusst getrennt unter:

- `../abhandlungen/`

Damit bleibt die Konstruktion schlank, waehrend laengere fachliche Texte
spaeter als eigener Mini-DIO-Projektbereich veroeffentlicht werden koennen.
