# 04 Memory

Memory speichert reale Erfahrung.

## Aufgabe

Memory verbindet reale Form, reale MCM-Wirkung und reale Konsequenz.

Memory ist der Speicher für reale Erfahrung, nicht für freie Gedanken.

## Darf

- Ähnlichkeit suchen
- Erfahrungsfamilien finden
- Nutzen-Spuren tragen
- Schmerz-/Belastungsspuren tragen
- Kontaktreife tragen

## Darf nicht

- freie Gedanken als Realität speichern
- Hypothesen als bewiesene Erfahrung behandeln
- Entry erzwingen

## Schreibvertrag

Memory darf nur schreiben, wenn ein Realitätsanker vorhanden ist:

- reale Form
- reale MCM-Wirkung
- reale Zeitlage
- reale Handlung oder Nicht-Handlung
- reale Konsequenz oder Beobachtungsfolge

Hypothesen dürfen nur als Bezug gespeichert werden, nicht als Wahrheit.
