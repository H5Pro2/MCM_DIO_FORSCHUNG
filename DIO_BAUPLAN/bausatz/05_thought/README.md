# 05 Thought

Thought bildet Gedanken, Varianten und Hypothesen.

Eine Hypothese ist Wahrnehmung des eigenen Denkens, keine Strategie:

```text
Ich dachte, diese Lage könnte so weiterlaufen.
Mein Gedanke hat getragen.
Mein Gedanke hat nicht getragen.
```

## Aufgabe

Thought erzeugt mögliche Deutungen aus Form, Feld und Memory.

Thought speichert Denkspuren getrennt vom realen Memory.

## Darf

- Varianten öffnen
- Gedankenkeime bilden
- Replay anstoßen
- Was-wäre-wenn-Spuren halten
- Hypothesen benennen
- Gedanken später an Realität und Konsequenz prüfen

## Darf nicht

- Realität ersetzen
- direkt Entry-Anker sein
- eigene Fantasie als Memory-Anker speichern
- Handlung erzwingen
- Strategie erzwingen

## Schreibvertrag

Thought-Memory speichert:

- Gedanke
- reale Form, auf die der Gedanke bezogen war
- MCM-Lage, aus der der Gedanke entstand
- spätere Bestätigung, Widerlegung oder Reorganisation

Ohne realen Anker bleibt ein Gedanke offen und darf nicht als Erfahrung
gespeichert werden.
