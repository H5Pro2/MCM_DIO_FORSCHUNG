# 02 Seeing

Seeing ist das Sinnesorgan für Form und Marktmelodie.

## Aufgabe

Seeing übersetzt Welt in Wahrnehmung:

- sichtbare Form
- hörbare Marktmelodie

## Darf

- Richtung sehen
- Verdichtung sehen
- Bereich sehen
- Wiederkehr sehen
- zeitliche Tiefe erkennen
- Marktenergie als Lautstärke/Frequenz hören
- Unsicherheit melden

## Darf nicht

- Entry erzeugen
- Trade-Richtung erzwingen
- menschliche Patternlabels als Wahrheit setzen
- MCM-Feld ersetzen

## Ausgang

```text
VisualFormState
MarketHearingState
```
