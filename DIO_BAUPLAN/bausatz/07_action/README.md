# 07 Action

Action ist reale Handlung oder bewusste Nicht-Handlung.

## Aufgabe

Action setzt eine Handlung nur dann um, wenn reale Nähe und Value-Prüfung
tragfähig sind.

Action setzt keine Strategie um, die vorher fest eingebaut wurde. Wenn
strategisches Verhalten sichtbar wird, dann als gereifte Reaktion aus
wiederkehrender Form, MCM-Wirkung, Thought-Prüfung und Konsequenz.

## Darf

- Entry-Kandidat aus realem Kontakt bilden
- SL/TP/Order-Geometrie prüfen
- Nicht-Handlung als Entscheidung führen
- Cancel oder Wait abbilden

## Darf nicht

- Impuls allein ausführen
- Hypothese allein ausführen
- innere Zustände als harte Sperren behandeln
- Strategie mechanisch erzwingen

## Hartes Gate

Nur das Value-Gate ist hart:

- gültiger Entry
- gültiger SL
- gültiger TP
- sinnvolles Risiko
- reale Orderfähigkeit
