# 12 Language

Language beschreibt DIOs eigene Formsprache.

## Aufgabe

Language verdichtet Form, Feld, Memory, Thought und Konsequenz zu eigener
Syntax.

## Darf

- Formzeichen bilden
- Satzspuren bilden
- semantische Herkunft prüfen
- Rohdaten kognitiv verdichten
- eigene Begriffe tragen

## Darf nicht

- menschliche Labels als Wahrheit setzen
- Gedanken als Realität tarnen
- Memory ersetzen

## Ausgang

```text
LanguageState
```
