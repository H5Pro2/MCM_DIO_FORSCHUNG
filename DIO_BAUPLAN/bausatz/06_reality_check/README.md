# 06 Reality Check

Reality Check trennt Gedanke und Realität.

Er prüft nicht, ob eine Strategie-Regel erfüllt ist, sondern ob ein Gedanke an
der realen Form und der MCM-Gefühlslage trägt.

## Aufgabe

Reality Check prüft, ob Thought zu Welt, Sinneswahrnehmung, MCM-Feld und
Memory passt.

## Darf

- Reife ableiten
- Vertrauen ableiten
- Vorsicht ableiten
- Reorganisation ableiten
- Realitätsbindung prüfen

## Darf nicht

- hart blockieren
- aus Hypothese direkt Handlung machen
- Memory überschreiben
- Strategie erzwingen

## Ausgang

```text
RealityCheckState
```
