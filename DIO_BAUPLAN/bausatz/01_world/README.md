# 01 World

World ist die reale Außenwelt.

## Aufgabe

World nimmt Rohdaten auf und erzeugt `WorldState`.

## Darf

- Zeitstempel tragen
- Rohwerte tragen
- reale Reihenfolge tragen
- aktuelle Weltposition tragen

## Darf nicht

- Muster benennen
- Handlung vorschlagen
- gut/schlecht bewerten
- MCM-Wirkung vorwegnehmen

## Ausgang

```text
WorldState
```
