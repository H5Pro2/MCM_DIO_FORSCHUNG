# 11 Neurochemistry

Neurochemistry beschreibt technische neuromodulatorische Achsen.

## Aufgabe

Neurochemie moduliert Feld, Neuronen, Regulation und Handlungstiefe.

Sie liegt zwischen gefühlter Feldwirkung und regulierter Haltung:

```text
MCM Field -> Neurochemistry -> Regulation
```

## Darf

- Aktivierung modulieren
- Schutzspannung modulieren
- Stabilität modulieren
- Hemmung modulieren
- Lernspannung modulieren

## Darf nicht

- als Trade-Signal wirken
- als hartes Gate wirken
- gut/schlecht erzwingen
- MCM-Feld ersetzen

## Ausgang

```text
NeurochemicalState
```
