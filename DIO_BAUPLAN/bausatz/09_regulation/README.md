# 09 Regulation

Regulation hält DIO handlungsfähig.

## Aufgabe

Regulation moduliert Haltung, Fokus, Abstand, Last und Erholung.

Sie liest das MCM-Feld, ist aber nicht das MCM-Feld.

```text
MCM Field = So wirkt es in mir.
Regulation = So gehe ich damit um.
```

## Darf

- Fokus erhöhen oder senken
- Abstand erzeugen
- Nervensystemlast abbilden
- kognitive Last abbilden
- Reflexion anziehen
- Erholung anfordern

## Darf nicht

- Realität verändern
- harte innere Gates setzen
- MCM-Feld als Gefühl ersetzen
- Thought, Memory oder Action ersetzen

## Ausgang

```text
RegulationState
```
