# 08 Consequence

Consequence ist die Rückwirkung der Realität.

## Aufgabe

Consequence bewertet, was aus Handlung, Nicht-Handlung oder Beobachtung wurde.

Sie führt Rückwirkung getrennt zurück:

- reale Erfahrung in Memory
- Gedankenbewertung in Thought-Memory
- innere Wirkung in MCM-Feld
- Umgang damit in Regulation

## Darf

- TP/SL/Cancel/Timeout einordnen
- verpasste Handlung einordnen
- Beobachtungserfolg einordnen
- Positionslast zurückführen
- Prozessqualität tragen

## Darf nicht

- nur Gewinn gut und Verlust schlecht setzen
- Thought nachträglich als Realität fälschen
- harte Zukunftsregeln erzeugen

## Regel

Consequence darf sagen:

```text
Mein Gedanke hat getragen.
Mein Gedanke hat nicht getragen.
```

Sie darf nicht sagen:

```text
Mein Gedanke war automatisch Realität.
```
