# MCM Trading Brain

MCM Trading Brain ist der Arbeitsraum für DIO.

## Begriffe

- **MCM** = **[Mental Core Matrix](https://github.com/H5Pro2/Mental-Core-Matrix-MCM)**  
  Der MCM-Begriff beschreibt hier den inneren Spannungs-, Wahrnehmungs-,
  Erfahrungs- und Regulationsraum.

- **DIO** = **[Digitaler Organismus](DIO_BAUPLAN/konstruktion/00_BAUPLAN.md)**  
  DIO ist das konkrete System, das mit Weltwahrnehmung, MCM-Feld, Memory,
  Thought, Neurochemie, Handlung, Konsequenz und Regulation arbeitet.

## Grundrichtung

Trading ist die erste Testwelt, nicht der eigentliche Kern.

DIO soll kein klassischer Signal-Bot mit starren Regeln sein. Ziel ist ein
fühlender, selbstregulativer digitaler Organismus:

```text
Welt
-> Sinneswahrnehmung
-> MCM-Feld
-> Memory
-> Thought
-> Realitätsprüfung
-> Handlung
-> Konsequenz
-> Regulation
```

Wichtig:

- Das MCM-Feld ist Gefühlswahrnehmung, nicht Strategielogik.
- Hypothesen sind Gedanken, nicht Realität.
- Strategie wird nicht erzwungen, sondern darf als tragende Reaktion aus
  Erfahrung entstehen.
- Harte innere Gates werden vermieden; nur das Value-Gate bleibt technische
  Pflichtprüfung für gültige Order-Geometrie.
- Debug ist Beobachtung, nicht Mechanik.
